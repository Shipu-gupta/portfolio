Mini Calculator App

About This Project

This mini calculator is built using HTML, CSS, and JavaScript. It supports addition, subtraction, multiplication, division, square, square root, and percentage operations.

Features

  • Add two numbers
  • Subtract two numbers
  • Multiply two numbers
  • Divide two numbers
  • Square (x²)
  • Square Root (√x)
  • Percentage (%)
